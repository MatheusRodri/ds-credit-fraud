# Analise de fraude de crédito

Esse projeto faz parte do matéria de paradigmas da programação do 6º semestre, de Sistemas da Informação do Centro Universitario Senac, na qual poderia ser feita uma pesquisa de algumas empresas e como elas utlizam Big Data e IA ou desenvolver um projeto usando Big Data e IA, e o nosso grupo resolveu o projeto. 

O projeto se trata de análise de dados de uma base de fraude de cartão de crédito, na qual podemos treinar e fazer um teste na base.
